شرح وسوم <head>

<meta charset="UTF-8">
يحدد ترميز النصوص ويضمن دعم الأحرف العربية.
<meta name="viewport">
يجعل الموقع متجاوبًا مع جميع الأجهزة.
<meta name="description">
يعطي وصفًا مختصرًا للصفحة يساعد في تحسين ترتيبها بمحركات البحث.
<meta name="keywords">
يحدد الكلمات المفتاحية التي تصف محتوى الصفحة لمحركات البحث.
<meta name="author">
يحدد اسم مؤلف الموقع.
<meta name="robots">
يوجه محركات البحث حول كيفية أرشفة الصفحة.
<meta property="og:title">
عنوان الصفحة عند مشاركتها على وسائل التواصل الاجتماعي.
<meta property="og:description">
وصف يظهر عند مشاركة الرابط.
<meta property="og:image">
رابط الصورة التي تظهر عند مشاركة الصفحة.
<meta property="og:url">
الرابط الأساسي للصفحة عند مشاركتها.
<link rel="icon">
يحدد أيقونة الصفحة (Favicon).
<link rel="stylesheet">
يربط ملف CSS خارجي لتنسيق الصفحة.
<script>
يربط ملف JavaScript خارجي لتنفيذ وظائف إضافية.
